## Data Privacy Compliance Checklist

### Data Inventory
- [ ] Personal data processing activities mapped
- [ ] Data flows documented (collection, storage, transfer)
- [ ] Legal basis for each processing activity identified
- [ ] Retention periods defined and enforced
- [ ] Third-party processors identified and assessed

### Individual Rights
- [ ] Privacy notice published and accessible
- [ ] Consent mechanisms implemented where required
- [ ] Subject access request process established
- [ ] Deletion/erasure capability implemented
- [ ] Data portability supported

### Security & Breach Response
- [ ] Technical security measures implemented
- [ ] Access controls and authentication
- [ ] Encryption at rest and in transit
- [ ] Breach detection and notification process
- [ ] Incident response plan tested

### Governance
- [ ] Data Protection Officer appointed (if required)
- [ ] Privacy impact assessments conducted
- [ ] Employee training completed
- [ ] Vendor due diligence process
- [ ] Regular compliance audits scheduled

## Incident Response Protocol

When a potential compliance incident occurs:

1. **Assess** - Determine scope and severity
2. **Contain** - Stop ongoing harm
3. **Document** - Preserve evidence
4. **Notify** - Inform required parties (regulators, individuals)
5. **Remediate** - Fix root cause
6. **Report** - Document lessons learned

### Notification Timelines

| Regulation | Authority Notification | Individual Notification |
|------------|----------------------|------------------------|
| GDPR | 72 hours | Without undue delay |
| CCPA | Reasonable time | Most expedient time |
| HIPAA | 60 days | 60 days |
| State Breach Laws | Varies (24h - 90 days) | Varies |
