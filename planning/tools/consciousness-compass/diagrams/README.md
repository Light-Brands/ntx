## Reading the Diagram

### Left to Right (Time)
```
PAST (Reflection) → PRESENT (Perception) → FUTURE (Projection)
```

### Top to Bottom (Energy)
```
CREATE (Conscious, Give) → CONSUME (Unconscious, Receive)
```

### Center Outward (Manifestation)
```
Heart/Mind Integration → Desires → Faith → Consciousness → Decisions
```

### Diagonals (Internal Dialogue)
```
Positive → Life/Joy/Love (upper left)
Negative → Death/Fear/Pain (lower left)
```
