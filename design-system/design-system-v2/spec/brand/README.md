# Brand Identity Specifications

Templates for defining brand identity across the platform.

## Templates

| Template | Purpose |
|----------|---------|
| [visual-identity-template.md](visual-identity-template.md) | Colors, typography, spacing |
| [brand-voice-template.md](brand-voice-template.md) | Tone, messaging, copy guidelines |
| [ai-personality-template.md](ai-personality-template.md) | AI assistant personality guide |
| [brand-integration-template.md](brand-integration-template.md) | How brand informs product |

## Brand Identity Pillars

### Visual Identity
- Color palette with semantic meanings
- Typography hierarchy
- Spacing and layout system
- Component styling guidelines

### Brand Voice
- Tone and personality
- Writing guidelines
- Error message patterns
- Empty state copy

### AI Personality
- Conversational style
- Response patterns
- Emotional intelligence guidelines
- Boundary handling

## Usage

1. Copy templates to `brands/{brand-name}/spec/brand/`
2. Complete each section with brand-specific content
3. Cross-reference with design system tokens
4. Train AI agents on personality guide
