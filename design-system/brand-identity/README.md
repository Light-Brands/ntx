# VIBEUP Brand Identity

## Overview

This folder contains the complete brand identity system for VIBEUP—the world's first social wellness platform. These documents define how we look, sound, and feel across every touchpoint.

## Brand Principles Summary

### Visual
- Minimal, calm, intentional
- Deep blues for trust and grounding
- Whites and neutrals for clarity
- Accents for energy and guidance

### Voice
- Calm confidence
- Warm empathy
- Conscious empowerment
- Minimal and intentional
- Modern, elevated, accessible

### Product
- Build for nervous systems, not dopamine loops
- Design for presence, not distraction
- Optimize for connection, not consumption
- Elevate energy, deepen connection, enable healing

### Mira
- Calm, wise, supportive, empathic
- Questions over commands
- Observations over judgments
- Invitations over instructions

## Governance

### Updates

All brand identity updates should:
1. Be documented in the relevant file
2. Be reviewed by brand lead
3. Be communicated to design and product teams
4. Maintain consistency with core principles

### Questions

For brand identity questions:
- **Visual/Design:** Reference `01-visual-identity.md`
- **Copy/Voice:** Reference `03-brand-voice-messaging.md` or `04-mira-personality-guide.md`
- **Product/Tech:** Reference `05-product-vision.md`
- **UX/Flows:** Reference `06-ux-wireframe-blueprint.md`
